# swiftui-practice

Welcome to the **swiftui-practice** repo! This is where I store all the exercises, projects, and examples while learning SwiftUI. This repository tracks my learning progress with various SwiftUI concepts, from basic to advanced techniques.

## Objective

The goal of this project is to practice and master SwiftUI by building small apps, exploring new components, and learning the essential principles of the framework, including:
- Building user interfaces (UI) using SwiftUI components.
- Managing state, data, and interactions in SwiftUI.
- Creating responsive and adaptive UIs for different iOS devices.
- Developing simple and more complex applications to improve iOS development skills.
